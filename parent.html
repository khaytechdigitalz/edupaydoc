How to Start EDUPAY POINT

Login to Parent Panel

For login to admin panel please follow the next steps: 
1. Run the project path from any web browser like (http://www.yourdomain.com/). 
2. Then you will see the following Login Screen.
3. Enter your username and password.
4. Then Click to "Log In" Button.
5. You are done with login.

Lost/Reset Password

Lost/Reset Password & Recover

For Lost password & recover password please follow the next steps: 
1. From login screen please click Lost Password 
2. Then you will see another Screen.
3. Enter your user email
4. Then Click to "Send" Button.
5. After send you will get Email with password reset/recovery instructions.
5. You are done with Lost & Recover Password.

Dashboard

View Dashboard Statistics

For view dashboard statistics please follow the next steps: 
1. Run the project path from any web browser like (http://www.yourdomain.com/). 
2. Then you will see the following Login Screen.
3. Enter your username and password.
4. Then Click to Log In Button.
5. After successfull login you will redirect to Parent Dashboard 
6. You are done with View Dashboard.

To Pay Fee/Invoice please follow the next steps:

Online Exam

1. Exam Instruction

For View Exam Instruction please follow the next steps: 
1. Login to Student Panel.
2. Select Exam Instruction sub menu under Online Exam main menu from left side bar.

2. Take Exam [Student Only]

For Manage Take Exam please follow the next steps: 
1. Login to Student Panel.
2. Select Take Exam sub menu under Online Exam main menu from left side bar.
3. To Take Exam from this exam list click Start Exam Button  Then Follow the next step and Start Exam.
4. You are done with Manage Take Exam feature.

5. Exam Result

For Manage Exam Result please follow the next steps: 
1. Login to Student Panel.
2. Select Exam Result main menu from left side bar.

4. Then click Find  button to get Exam Result list
6. You are done with Exam Result feature.

Profile

1. Manage My Profile

For Manage My Profile please follow the next steps: 
1. Login to System Admin Panel.
2. Select My Profile sub menu under Profile main menu from left side bar.
3. To Update Profile click Update Tab and click Update Button at the bottom.
4. You are done with Manage My Profile feature.

2. Reset Password

For Reset Password please follow the next steps: 
    . Select Reset Password sub menu under Profile main menu from left side bar.
3. To Reset Password  input your new password in the fields and click Submit Button.
4. You are done with Reset Password feature.

3. Logout

For Logout please follow the next steps: 
1. Login to System Admin Panel.
2. Click Logout sub menu under Profile main menu from left side bar.
3. You are done with Logout feature.
